WatchTonight publishes a curated movie list in the StevenLu format, fully compatible with Radarr's built-in list import. Follow the steps below to auto-populate your Radarr watchlist with everything we're tracking tonight.

Step-by-step guide
Adding the list in Radarr
1

Open Radarr and go to Settings

In your Radarr instance, click Settings in the left sidebar.

2

Navigate to Lists

Select the Lists tab inside Settings.

3

Add a new list

Click the + button to add a new list source.

4

Choose "StevenLu's Popular Movies"

From the list type picker, scroll to and select StevenLu's Popular Movies. This type accepts any JSON feed that follows the StevenLu format — which is exactly what WatchTonight exports.

5

Paste the WatchTonight feed URL

In the URL field, enter:

https://watchtonight.app/radarr.json

6

Configure quality & folder settings

Set your preferred Quality Profile, Root Folder, and monitoring options (e.g. Monitor: Movie Only or Monitor and Add).

7

Save and sync

Click Save. Radarr will begin importing the list on its next scheduled sync, or you can trigger a manual sync from Movies → Discover → Add Missing.

About the StevenLu list format

The StevenLu format is a simple JSON array where each entry contains a movie title and an optional IMDB ID. Radarr uses this to match and add titles automatically. Our feed is updated daily alongside the WatchTonight curated picks.

For full details on how Radarr handles this list type, see the official docs: wiki.servarr.com → Radarr Supported → StevenLu Import
