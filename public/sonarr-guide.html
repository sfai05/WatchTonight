Sonarr Integration

Import our list
into Sonarr

WatchTonight publishes a curated TV series list in a TVDB-based JSON format that works with Sonarr's custom import list feature. Follow the steps below to automatically track tonight's picks in your Sonarr library.

The list URL
JSON feed https://watchtonight.app/sonarr.json
Step-by-step guide
Adding the list in Sonarr
1

Open Sonarr and go to Settings

In your Sonarr instance, click Settings in the left sidebar.

2

Navigate to Import Lists

Select the Import Lists tab inside Settings.

3

Add a new import list

Click the + button to add a new list source.

4

Choose "Custom List"

From the list type picker, select Custom List. This type accepts a JSON feed containing TVDB IDs — the same format WatchTonight exports.

5

Paste the WatchTonight feed URL

In the URL field, enter:

https://watchtonight.app/sonarr.json

6

Configure quality & folder settings

Set your preferred Quality Profile, Root Folder, Series Type (Standard / Anime / Daily), and monitoring options (e.g. Monitor: All Episodes or Future).

7

Save and sync

Click Save. Sonarr will import the list on its next scheduled sync, or you can trigger a manual sync from Series → Add New → search for a title from the list, or via System → Tasks → Refresh Import Lists.

About the list format

Our Sonarr feed is a JSON array of objects, each containing a tvdbId — the unique identifier used by TheTVDB and Sonarr to match series. The list is refreshed daily alongside the WatchTonight curated picks.

For full details on Sonarr's import list feature, see the official docs: wiki.servarr.com → Sonarr Settings → Import Lists
